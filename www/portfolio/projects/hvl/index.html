Home Value Leads

 UI/UX Design, Prototyping View Work

Home Value Leads is a SAAS lead-generation tool for real estate agents. Agents pay a monthly fee to set up websites that capture email addresses and generate home value reports for people interested in selling their home. I redesigned the software tool, created prototypes for new feature sets, designed a suite of HTML emails, and created a new landing page to increase signups. Throughout the process I used many tools/techniques including Sketch, Invision and Principal for prototyping, and code (HTML + CSS + JS).

Currently we are going through a phase of A/B testing with HVL and trying to optimize the product. The whole project from intial concepting to final implementation took about 9 months.
